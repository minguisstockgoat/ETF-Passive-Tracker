# 로컬 대시보드 실행 안내 (Windows)

패시브 ETF 구성비중 트래커(SOL·RISE·TIGER 20종)를 내 PC에서 실행하는 방법.
과거 이력이 담긴 `data\etf.db` 가 함께 들어있어 **설치 직후 바로 전일 대비 비교**가 보입니다.

---

## 1. 준비 (최초 1회)

1. **Python 설치** — https://www.python.org/downloads/ 에서 설치.
   설치 첫 화면에서 **`Add python.exe to PATH` 체크** 꼭 하세요.
2. 이 압축을 원하는 폴더에 풉니다.
3. 압축 푼 폴더(= `server.py` 있는 폴더)의 **주소창에 `cmd` 입력 후 Enter** → 그 폴더에서 명령창이 열립니다.
   아래 한 줄로 필요한 패키지 설치:
   ```
   py -m pip install -r requirements.txt
   ```
   (`requests`, `beautifulsoup4` 두 개만 설치됩니다. 나머지는 파이썬 기본 포함.)

---

## 2. 실행

- **`run_dashboard.bat` 더블클릭** → 검은 창이 뜨고 브라우저가 http://127.0.0.1:8850 를 자동으로 엽니다.
- 그 검은 창을 **닫지 마세요** (닫으면 서버도 꺼짐). 끌 때는 창에서 `Ctrl+C` 또는 창 닫기.
- 인터넷 없이도 **보기**는 됩니다 (DB가 이미 들어있음).

---

## 3. 데이터 최신화 (선택)

- **`update.bat` 더블클릭** (또는 명령창에서 `py ingest.py --latest`)
  → 최신 영업일 구성종목을 받아 DB에 추가합니다.
- 인터넷만 있으면 되고, **API 키·로그인 불필요** (전부 운용사 공개 데이터).
- 매일 자동으로 돌리려면 Windows **작업 스케줄러**에 `update.bat` 를 평일 아침으로 등록하면 됩니다.

---

## 4. 폴더 / 수정 포인트

| 바꾸고 싶은 것 | 파일 |
|---|---|
| 대상 ETF 목록·강조 임계치 | `config.py` |
| 화면 디자인/레이아웃 | `web\index.html`, `web\style.css`, `web\app.js` |
| 운용사별 수집기 | `fetchers.py` |
| 일간 변화(리밸런싱) 계산 | `diff.py` |
| 데이터 저장소(SQLite) | `data\etf.db` |

> 정적 배포(GitHub Pages)로도 만들 수 있습니다: `py build_static.py` → `docs\` 폴더 생성 → 그 폴더를 Pages로 서빙.

---

## 문제 해결

- `py` 명령이 안 되면 `python` 으로 바꿔 실행: `python server.py`
- **8850 포트 충돌** 시 `config.py` 의 `SERVER_PORT` 숫자를 바꾸세요.
- 첫 실행 때 방화벽 팝업이 뜨면 **허용**.
- 데이터를 처음부터 다시 쌓고 싶으면: `py ingest.py` (과거 영업일 백필까지 수행, 몇 분 소요).
